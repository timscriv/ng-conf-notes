# Progressive Web Apps

- The Web has more reach
    - Searchable
    - Shareable
    - Low Friction
- Why Native Apps?
    - Performant
    - Re-engagement
    - API Access (GPS, Bluetooth, etc.)
- Progressive Web Apps
    - Instant Loading
        - App Shell
    - Offline
        - Service Worker
            - Can cache content without a network connection
    - Installable
        - Web App Manifest
            - Meta data about the application
    - Notifications
